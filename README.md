# 🌟 Aaradhya AI — Your Personal Desktop Assistant

Aaradhya AI is a premium, interactive desktop assistant built with Python and CustomTkinter, powered by the cutting-edge **Google Gemini Live API**. It features real-time voice interaction, fluid animations, customizable themes, and a modern desktop widget interface.

---

## ✨ Features

*   🗣️ **Real-Time Voice Chat:** Speak naturally with Aaradhya AI and listen to high-quality audio responses.
*   🎨 **Beautiful Visuals:** A sleek glassmorphic desktop interface with smooth micro-animations.
*   🎭 **Custom Personalities:** Switch between different response behaviors (Assistant, Friendly, Professional).
*   🌈 **Theme Customization:** Custom accent colors and pre-configured themes (Dark Purple, Midnight Blue, Rose Gold, and more).
*   ⚙️ **Custom Configuration:** Personalize your wake word, microphone settings, and model choices directly from the settings panel.

---

## 🚀 How to Download & Run

1.  On the right-side menu of this repository, click on **Releases** (or download the latest release directly).
2.  Download the **`Aaradhya_AI.zip`** asset.
3.  Extract the ZIP file to a folder on your computer.
4.  Open the extracted folder and double-click **`Aaradhya_AI.exe`** to start Aaradhya AI!

---

## 🔑 How to Setup Gemini API Key

Aaradhya AI requires a Gemini API Key to connect to Google's AI models. Getting a key is free and takes less than a minute:

1.  Visit **[Google AI Studio](https://aistudio.google.com/)**.
2.  Sign in using your Google account.
3.  Click the blue **Get API key** button in the top left corner.
4.  Click **Create API key**, select a project (or create one), and copy your generated key.
5.  Launch `Aaradhya_AI.exe`, paste your key into the **Onboarding Wizard** or **Settings Panel**, and you're ready to chat!
